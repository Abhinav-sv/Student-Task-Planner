# Student Task Planner

## Description

Student Task Planner is a web application that helps students organize and manage their daily tasks efficiently.

Users can create tasks with a title, description, and due date. The application provides a simple interface to add and manage tasks, helping students improve productivity and time management.

## Features

- Add tasks
- View tasks
- Delete tasks
- Simple and clean user interface
- Backend setup using Node.js and Express.js

## Technologies Used

Frontend:
- HTML
- CSS
- JavaScript

Backend:
- Node.js
- Express.js

## How to Run the Project

### Frontend

Open the `index.html` file in your browser.

### Backend

Open the terminal inside the backend folder:

```bash
cd backend
```

Install dependencies:

```bash
npm install
```

Start the server:

```bash
npm start
```
